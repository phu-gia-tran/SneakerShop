# Frontend Reference

Thu muc nay chi giu source frontend ban dau de doi chieu.

Project ASP.NET MVC dang chay nam trong `../BE`, va frontend da duoc dua vao dung vi tri MVC:

- `../BE/Views`
- `../BE/Assets`
- `../BE/Content`
- `../BE/Scripts`
- `../BE/fonts`

Khi mo project bang Visual Studio, hay mo file solution o thu muc goc:

```text
../Sneaker.sln
```

Khong dung `FE/Sneaker.reference.sln` lam solution chinh.
